### the difference between window , document , consolo ?

## 1.document:
 The document object represents the HTML page (DOM) loaded in the browser.

It is used to access and manipulate HTML elements

## 2.window:
The window object represents the browser window.

It is the top-level (global) object in the browser.

## 3. consolo:
The console object is used for debugging.

It prints messages in the browser console.

Mainly used by developers, not visible to users.
